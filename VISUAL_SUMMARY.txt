┌─────────────────────────────────────────────────────────────────────────────┐
│ 🎓 ANTES vs DEPOIS                                                          │
└─────────────────────────────────────────────────────────────────────────────┘

KEYS EM LISTAS:
  ❌ ANTES: {items.map((item, i) => <Item key={i} />)}
  ✅ DEPOIS: {items.map((item) => <Item key={`item-${item.id}`} />)}

HARDCODED VALUES:
  ❌ ANTES: padding: 12px; color: var(--texts);
  ✅ DEPOIS: padding: ${THEME.spacing.md}; color: ${THEME.colors.text};

COMPONENTES:
  ❌ ANTES: export default MyComponent
  ✅ DEPOIS: export default memo(MyComponent)

PROP TYPES:
  ❌ ANTES: const Button = ({ label }) => <button>{label}</button>
  ✅ DEPOIS: Button.propTypes = { label: PropTypes.string.isRequired }

NAVEGAÇÃO:
  ❌ ANTES: <Link to="/about">About</Link> × 5 vezes
  ✅ DEPOIS: {NAV_ITEMS.map(...)} - uma vez, reutilizável
